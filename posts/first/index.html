Hello World

This is my first post from my new blog. It look pretty good even if it’s only an early stage.

Why I built it

I found it a very useful tool for talking about myself, my passions and my worries.

I have a strong anxiety issue, so I think this blog can help me to feel better and beat this “Monster”.
I would like to clarify that I haven’t been diagnosed with any kind of panic disorder, I’m only trying to find a way to beat or limit my anxiety.

Maybe I need to look for a diagnose but I’m scared about the outcome.

Don’t be scared, I will also post about my passions like reading, software development, videogames and NBA.

How I built it

I make this blog using HUGO, a static website generator.

HUGO is built using one of my favorite programmaing language: Golang an awesome programming language by Google.

Before starting with Hugo I also considered Jekyll as my main platform for developing this blog. I drop it because I don’t know ruby enough for being productive.

On the other hand I like Golang so much, and before dive into this project I stepped up my knowledge of it by taking two online courses.
